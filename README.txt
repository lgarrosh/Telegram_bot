Цель проекта:
    Создать Telegram-бота, который помогает пользователям быть в курсе ситуации на рынке криптовалют и фиатных валют. Бот должен предоставлять ежедневные и запрашиваемые отчеты, а также аналитическую информацию, с возможностью гибкой настройки отчетности.

Основные функции:
    Ежедневные обновления цен:
        Бот отправляет каждое утро пользователю актуальные цены на выбранные криптовалюты и курсы фиатных валют.
        Возможность выбора конкретных криптовалют и фиатных валют для мониторинга.
    
    Запрос информации:
        Пользователь может в любое время запросить:
            Основную информацию по текущему состоянию рынка.
            Подробную информацию по выбранной криптовалюте (текущая цена, объем торгов, рыночная капитализация, изменения за 24 часа и т.д.).
            Аналитический отчет с прогнозом и трендами на основе текущих данных.
    
    Анализ рынка:
        Возможность получить аналитику по выбранной криптовалюте, включая новости, текущие тенденции и прогнозы.
    
    Настройка отчетности:
        В интерфейсе бота пользователь может настраивать параметры ежедневных отчетов:
            Время отправки отчета.
            Валюты и криптовалюты для мониторинга.
            Частота отправки аналитических отчетов.

Интерфейс:
    Удобный и интуитивно понятный интерфейс взаимодействия через Telegram:
        Команды с понятной структурой для пользователя.
        Гибкие настройки через встроенные кнопки и меню.

Технические требования:
    Среда разработки:
        Язык: Python.
        Фреймворк: python-telegram-bot.
        API для получения данных: использование публичных API криптовалютных бирж cryptocompare и API для фиатных валют (например, OpenExchangeRates).

    Интеграции:
        Взаимодействие с API для получения актуальной информации по криптовалютам и фиатным валютам.
        Возможность интеграции с внешними аналитическими системами или новостными источниками для получения данных и анализа.
    
    Безопасность:
        Безопасное хранение данных пользователей.
        Реализация методов защиты от несанкционированного доступа к боту.
    
    Масштабируемость:
        Возможность добавления новых функций и расширения списка поддерживаемых валют и источников данных без значительных изменений в архитектуре.

Дополнительные функции:
    Логирование запросов пользователя.
    Оптимизация использования API, чтобы не превысить лимиты запросов.
    Возможность получения push-уведомлений о значительных изменениях на рынке (например, резкий рост или падение цены).

Важные моменты:
    Бот должен работать быстро и без сбоев.
    Удобство использования и интуитивная настройка отчётов пользователем — ключевой фактор.
